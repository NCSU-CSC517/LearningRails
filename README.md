# Learning Rails
This repository is mainly for CSC/ECE 517 class use at the NCSU.

#### Description
The repository contains combined contribution of individual students who developed Java and Ruby in-class programming exercises, problems, and examples that are part of the CSC/ECE 517 (Object-Oriented Design and Development) curriculum. It contains multiple repositories that were mirrored from different locations.

#### Source Code
This section describes how and from where some of the source code has been taken.
##### Mirrored Repositories
Some directories located in this repository were mirrored from their original location using the following git command:
```sh
$ git clone --mirror https://github.com/username/repo.git
```
The push location of all the mirror was set using the following commands:
```sh
$ cd repo.git
$ git remote set-url --push origin https://github.com/NSCU-CSC517/LearningRails.git
```
After the push location was set, the source code was pushed into this repository.
Below is the list of directories that were mirrored as described, their authors and original source repository:

    - *Author:* Brent Younce
    - *Originally Mirrored:* https://github.com/BrentYounce/517Exercises.git
